# Components Directory

This directory contains reusable React Native components that can be used across multiple screens.

## Examples of components to add:

- **Button.js** - Custom button component
- **Card.js** - Card component for displaying content
- **Input.js** - Custom text input component
- **Header.js** - Reusable header component
- **LoadingSpinner.js** - Loading indicator component
- **TestCard.js** - Component for displaying test series information
- **QuestionCard.js** - Component for displaying questions
- **ScoreCard.js** - Component for displaying scores and results

## Usage

```javascript
import CustomButton from '../components/Button';

<CustomButton 
  title="Submit" 
  onPress={handleSubmit}
  variant="primary"
/>
```
